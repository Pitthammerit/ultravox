## 1. Problem statement

Today every transcription roundtrips audio (typically 30–90 KB Opus) to the Cloudflare Voice Worker, which calls Workers AI Whisper-large-v3-turbo, then optionally calls a cleanup LLM. We want a **local-only transcription path** that runs Whisper on-device using Apple Silicon's Metal/Neural-Engine compute, mirroring the Superwhisper default path.

**Why now:**

1. **Speed.** A warm in-process Whisper-base/small on M-series typically transcribes a 10 s clip in ~300–700 ms (Metal). The Worker round-trip is dominated by 200–800 ms of network + cold-start, not compute. For short dictations the local path will feel near-instant.
2. **Cost / token usage.** Each Worker transcription is ~$0.0005 (Workers AI billing). For a heavy dictator (100/day) that's ~$1.50/mo. Local is free at runtime.
3. **Privacy.** Audio never leaves the device. A real differentiator vs cleanup-LLM-on-cloud, which the user opted-in to separately.
4. **Offline capability.** Works on planes, in hotel Wi-Fi dead zones, in EU networks the user doesn't trust.

**Confirmation from Superwhisper inspection** (see `docs/research/superwhisper-architecture.md` and `superwhisper-pipeline-and-prompts.md`): Superwhisper's binary references both `argmaxinc/whisperkit-coreml` (CoreML/Neural Engine) and `https://huggingface.co/ggerganov/whisper.cpp/...ggml-*.bin` (whisper.cpp/Metal). Models are downloaded on-demand from HuggingFace; nothing is bundled in the 132 MB DMG. Default model on Superwhisper appears to be `openai_whisper-large-v3-v20240930_turbo_632MB`. We adopt the same on-demand pattern.

## 2. Out of scope

The following are **explicitly deferred** and must not be added to this plan:

1. **Cleanup LLM going local.** Stays cloud (Cloudflare worker → OpenRouter) or Claude Code CLI. Local on-device cleanup LLM via llama.cpp/Metal is a separate v0.11 plan.
2. **Auto-update mechanism for downloaded models.** v1 ships a fixed model list. If we publish a new recommended model, users redownload manually from Settings.
3. **Model picker in onboarding wizard.** Local Whisper is opt-in via Settings → Configuration → "Local transcription". Onboarding stays the same. New users get the cloud worker path by default.
4. **Streaming / VAD-segmented transcription.** v0.10 transcribes a finalized audio blob just like the worker path. No partial results, no voice-activity-driven segmentation.
5. **Quality benchmarks vs cloud.** We don't gate the release on WER measurements. We expose the option, label it "Beta" in UI, and let users compare subjectively.
6. **Multi-backend ASR registry** (Deepgram / ElevenLabs Scribe / etc.). The provider abstraction in this plan is `cloud-worker | local-whisper` only.
7. **Windows / non-macOS builds.** Local Whisper in v0.10 is macOS-only — we tag the new Rust module `#[cfg(target_os = "macos")]` and short-circuit the UI on other platforms.
8. **Background prewarming of the model on app start.** The first transcription after launch will absorb the model-load cost (~1–2 s for base, ~3–5 s for large-v3). Prewarm-on-launch is a v0.10.x polish.
9. **Distil-Whisper / Parakeet / non-OpenAI models.** Stick to OpenAI Whisper checkpoints. Distil-Whisper would be a model-list addition; not now.
10. **GPU/Neural Engine selection UI.** Use the library default (Metal). No "Use Neural Engine" toggle.

## 3. Recommended approach + rationale

We have three realistic integration options. Read all three before picking.

### Option A — WhisperKit Swift package via Swift FFI

- Library: https://github.com/argmaxinc/WhisperKit (MIT, Swift).
- CoreML + Neural Engine → fastest on Apple Silicon.
- Same library Superwhisper's binary symbol table references.
- **Cost:** Swift→C bridge needed. Either (a) wrap WhisperKit in a thin Swift static library, expose a C API via `@_cdecl`, link from Rust via `bindgen` + `cc`; or (b) ship a sidecar `whisperkit-cli` binary and `Command::new` it (same as the Claude Code pattern). Both add Xcode-project complexity to a Cargo build.
- Tauri bundling: must add the Swift library + CoreML.framework + AppleNeuralEngine.framework to the `.app`. Manageable but new pipeline territory.

### Option B — `whisper-rs` (Rust bindings to whisper.cpp)

- Library: https://github.com/tazz4843/whisper-rs (MIT, well-maintained, Rust binding to ggerganov/whisper.cpp).
- Pure Cargo dependency: `whisper-rs = "0.11"` with feature `metal` enables Metal compute shaders.
- No Xcode project, no Swift bridge. Builds inside the existing `cargo tauri build` pipeline.
- Performance on M-series: Metal path is **~70–80% as fast as Neural Engine**, fully sufficient for the speed claim against the cloud roundtrip.
- Model files are GGML `.bin` blobs from HuggingFace `ggerganov/whisper.cpp` (already what Superwhisper uses on its whisper.cpp path).
- Mature: 800+ stars, Discord, weekly releases.

### Option C — Sidecar CLI binary (`whisper.cpp` build of `main` or argmaxinc `whisperkit-cli`)

- Spawn subprocess like `claude_code.rs` does today.
- Easiest to wire (just `Command::new`).
- **Disqualifying:** model load cost incurred on every dictation (~1–2 s for base). Defeats the speed claim.
- Bundle bloat: shipping a 5–10 MB CLI plus its Metal shaders plus model loaders.

### Decision: **Option B — `whisper-rs`** for v0.10.0.

Rationale:

1. **Lowest integration risk.** Single Cargo dep. No Xcode project, no Swift bridge, no entitlement changes beyond what we already have (we already have `cs.allow-unsigned-executable-memory`, `cs.disable-library-validation`, and Hardened Runtime works with whisper.cpp's Metal kernels per upstream CI).
2. **In-process, model stays warm.** A `WhisperContext` lives for the app's lifetime in a Rust `Mutex<Option<...>>`. Disqualifies Option C.
3. **Performance is sufficient.** Metal on M-series is 4–8× realtime on small/base/large-v3-turbo. The user perceives "instant" once the network round-trip is gone.
4. **Library risk hedged.** If Metal performance disappoints in dogfooding, Option A (WhisperKit + Neural Engine) is a v0.10.x follow-up — we keep the same `LocalWhisper` Rust trait and swap the implementation.
5. **No Hub-auth complexity.** GGML files are public, no HuggingFace token needed. Direct `https://huggingface.co/ggerganov/whisper.cpp/resolve/main/<file>` downloads work.

```rust
//! Decode arbitrary recorder output (webm/mp4/wav/ogg) to 16 kHz mono f32
//! PCM, the format Whisper expects.
//!
//! MediaRecorder on macOS WKWebView typically produces audio/mp4 (AAC) or
//! audio/webm;codecs=opus. We hand the bytes to symphonia, ask for the first
//! audio track, and resample to 16 kHz mono.

use std::io::Cursor;

use symphonia::core::audio::{AudioBufferRef, Signal};
use symphonia::core::codecs::{DecoderOptions, CODEC_TYPE_NULL};
use symphonia::core::formats::FormatOptions;
use symphonia::core::io::MediaSourceStream;
use symphonia::core::meta::MetadataOptions;
use symphonia::core::probe::Hint;

const TARGET_SR: u32 = 16_000;

pub fn decode_to_mono_16k(bytes: Vec<u8>, mime: &str) -> Result<Vec<f32>, String> {
    let cursor = Cursor::new(bytes);
    let mss = MediaSourceStream::new(Box::new(cursor), Default::default());

    let mut hint = Hint::new();
    if mime.contains("mp4") || mime.contains("m4a") || mime.contains("aac") {
        hint.with_extension("m4a");
    } else if mime.contains("ogg") || mime.contains("opus") {
        hint.with_extension("ogg");
    } else if mime.contains("wav") {
        hint.with_extension("wav");
    } else if mime.contains("webm") {
        // Symphonia has no webm container demuxer as of 0.5.x. Caller must
        // ensure useRecorder negotiates audio/mp4 on macOS — see plan §6.1.
        return Err("webm not supported by symphonia; recorder must produce mp4".into());
    }

    let probed = symphonia::default::get_probe()
        .format(&hint, mss, &FormatOptions::default(), &MetadataOptions::default())
        .map_err(|e| format!("probe: {e}"))?;
    let mut format = probed.format;
    let track = format
        .tracks()
        .iter()
        .find(|t| t.codec_params.codec != CODEC_TYPE_NULL)
        .ok_or("no audio track")?;
    let track_id = track.id;
    let src_sr = track.codec_params.sample_rate.ok_or("missing sample_rate")?;
    let channels = track
        .codec_params
        .channels
        .map(|c| c.count())
        .unwrap_or(1);

    let mut decoder = symphonia::default::get_codecs()
        .make(&track.codec_params, &DecoderOptions::default())
        .map_err(|e| format!("decoder: {e}"))?;

    let mut samples_mono: Vec<f32> = Vec::new();
    loop {
        let packet = match format.next_packet() {
            Ok(p) => p,
            Err(symphonia::core::errors::Error::IoError(_)) => break, // EOF
            Err(e) => return Err(format!("packet: {e}")),
        };
        if packet.track_id() != track_id {
            continue;
        }
        let decoded = match decoder.decode(&packet) {
            Ok(d) => d,
            Err(symphonia::core::errors::Error::DecodeError(_)) => continue,
            Err(e) => return Err(format!("decode: {e}")),
        };
        // Convert to f32 + downmix to mono.
        match decoded {
            AudioBufferRef::F32(buf) => {
                let frames = buf.frames();
                for f in 0..frames {
                    let mut acc = 0.0f32;
                    for ch in 0..channels {
                        acc += buf.chan(ch)[f];
                    }
                    samples_mono.push(acc / channels as f32);
                }
            }
            AudioBufferRef::S16(buf) => {
                let frames = buf.frames();
                for f in 0..frames {
                    let mut acc = 0.0f32;
                    for ch in 0..channels {
                        acc += buf.chan(ch)[f] as f32 / 32768.0;
                    }
                    samples_mono.push(acc / channels as f32);
                }
            }
            AudioBufferRef::S32(buf) => {
                let frames = buf.frames();
                for f in 0..frames {
                    let mut acc = 0.0f32;
                    for ch in 0..channels {
                        acc += buf.chan(ch)[f] as f32 / 2_147_483_648.0;
                    }
                    samples_mono.push(acc / channels as f32);
                }
            }
            _ => return Err("unsupported sample format".into()),
        }
    }

    if src_sr == TARGET_SR {
        return Ok(samples_mono);
    }
    Ok(linear_resample(&samples_mono, src_sr, TARGET_SR))
}

/// Cheap linear resampler. Whisper is robust to ~moderate aliasing; this is
/// good enough for dictation. If quality regresses on noisy input, swap in
/// the `rubato` crate (sinc filter).
fn linear_resample(input: &[f32], src_sr: u32, dst_sr: u32) -> Vec<f32> {
    if input.is_empty() {
        return Vec::new();
    }
    let ratio = dst_sr as f64 / src_sr as f64;
    let out_len = (input.len() as f64 * ratio) as usize;
    let mut out = Vec::with_capacity(out_len);
    for i in 0..out_len {
        let src_pos = i as f64 / ratio;
        let idx = src_pos.floor() as usize;
        let frac = (src_pos - idx as f64) as f32;
        let s0 = input[idx.min(input.len() - 1)];
        let s1 = input[(idx + 1).min(input.len() - 1)];
        out.push(s0 + (s1 - s0) * frac);
    }
    out
}
```

#### Step 1.3 — Add `local_whisper.rs` skeleton

Create `apps/ultravox/src-tauri/src/local_whisper.rs`:

```rust
//! Local on-device Whisper transcription via whisper-rs (whisper.cpp + Metal).
//!
//! Mirror of `claude_code.rs`'s "spawn-and-fall-back-to-worker" pattern, but
//! with an in-process `WhisperContext` kept warm across recordings.

use std::path::PathBuf;
use std::sync::Mutex;

use serde::Serialize;
use tauri::{AppHandle, Manager};
use whisper_rs::{FullParams, SamplingStrategy, WhisperContext, WhisperContextParameters};

use crate::audio_decode::decode_to_mono_16k;

/// Cached context. We keep ONE model loaded at a time; switching models
/// drops the previous context. Lock duration during transcribe is the
/// length of the audio in real-time (whisper-rs is sync).
struct Cached {
    model_path: PathBuf,
    ctx: WhisperContext,
}

#[derive(Default)]
pub struct LocalWhisperState {
    cached: Mutex<Option<Cached>>,
}

#[derive(Serialize)]
pub struct LocalWhisperStatus {
    pub available: bool,
    pub installed_models: Vec<String>,
    pub models_dir: String,
}

/// `~/Library/Application Support/com.ultravox.dev/models/whisper/`.
pub fn models_dir(app: &AppHandle) -> Result<PathBuf, String> {
    let dir = app
        .path()
        .app_data_dir()
        .map_err(|e| format!("app_data_dir: {e}"))?
        .join("models")
        .join("whisper");
    std::fs::create_dir_all(&dir).map_err(|e| format!("mkdir: {e}"))?;
    Ok(dir)
}

#[tauri::command]
pub fn local_whisper_check(app: AppHandle) -> Result<LocalWhisperStatus, String> {
    let dir = models_dir(&app)?;
    let mut installed = Vec::new();
    if let Ok(entries) = std::fs::read_dir(&dir) {
        for e in entries.flatten() {
            if let Some(name) = e.file_name().to_str() {
                if name.ends_with(".bin") {
                    installed.push(name.trim_end_matches(".bin").to_string());
                }
            }
        }
    }
    Ok(LocalWhisperStatus {
        available: true,
        installed_models: installed,
        models_dir: dir.to_string_lossy().to_string(),
    })
}

#[tauri::command]
pub fn local_whisper_transcribe(
    app: AppHandle,
    state: tauri::State<'_, LocalWhisperState>,
    model_id: String,
    audio_bytes: Vec<u8>,
    mime: String,
    language: Option<String>,
) -> Result<String, String> {
    let dir = models_dir(&app)?;
    let model_path = dir.join(format!("{model_id}.bin"));
    if !model_path.is_file() {
        return Err(format!("model not installed: {model_id}"));
    }

    let mut guard = state.cached.lock().map_err(|e| format!("lock: {e}"))?;
    let need_load = match guard.as_ref() {
        Some(c) => c.model_path != model_path,
        None => true,
    };
    if need_load {
        let params = WhisperContextParameters::default();
        let ctx = WhisperContext::new_with_params(
            model_path.to_str().ok_or("non-utf8 path")?,
            params,
        )
        .map_err(|e| format!("load model: {e}"))?;
        *guard = Some(Cached { model_path: model_path.clone(), ctx });
    }
    let cached = guard.as_ref().unwrap();

    let pcm = decode_to_mono_16k(audio_bytes, &mime)?;
    if pcm.is_empty() {
        return Err("decoded audio empty".into());
    }

    let mut state_w = cached.ctx.create_state().map_err(|e| format!("state: {e}"))?;
    let mut params = FullParams::new(SamplingStrategy::Greedy { best_of: 1 });
    params.set_n_threads(num_cpus());
    params.set_translate(false);
    params.set_no_context(true);
    params.set_single_segment(false);
    params.set_print_special(false);
    params.set_print_progress(false);
    params.set_print_realtime(false);
    params.set_print_timestamps(false);
    if let Some(lang) = language.as_deref() {
        if lang != "auto" {
            params.set_language(Some(lang));
        } else {
            params.set_language(None);
        }
    } else {
        params.set_language(None);
    }

    state_w.full(params, &pcm).map_err(|e| format!("full: {e}"))?;
    let n_segments = state_w.full_n_segments().map_err(|e| format!("n_segments: {e}"))?;
    let mut text = String::new();
    for i in 0..n_segments {
        let seg = state_w.full_get_segment_text(i).map_err(|e| format!("seg {i}: {e}"))?;
        text.push_str(&seg);
    }
    Ok(text.trim().to_string())
}

fn num_cpus() -> std::os::raw::c_int {
    std::thread::available_parallelism()
        .map(|n| n.get() as i32)
        .unwrap_or(4)
}
```

#### Step 2.2 — Add download command

In `apps/ultravox/src-tauri/src/local_whisper.rs`, append:

```rust
use sha2::{Digest, Sha256};
use std::io::Write as IoWrite;
use tauri::Emitter;
use futures_util::StreamExt;

#[derive(serde::Serialize, Clone)]
pub struct DownloadProgress {
    pub model_id: String,
    pub downloaded: u64,
    pub total: u64,
    pub done: bool,
    pub error: Option<String>,
}

#[tauri::command]
pub async fn local_whisper_download(
    app: AppHandle,
    model_id: String,
    url: String,
    expected_sha256: String,
) -> Result<(), String> {
    let dir = models_dir(&app)?;
    let final_path = dir.join(format!("{model_id}.bin"));
    let tmp_path = dir.join(format!("{model_id}.bin.part"));

    let client = reqwest::Client::builder()
        .timeout(std::time::Duration::from_secs(60 * 30))
        .build()
        .map_err(|e| format!("client: {e}"))?;
    let res = client.get(&url).send().await.map_err(|e| format!("get: {e}"))?;
    if !res.status().is_success() {
        return Err(format!("http {}", res.status()));
    }
    let total = res.content_length().unwrap_or(0);

    let mut file = std::fs::File::create(&tmp_path).map_err(|e| format!("create: {e}"))?;
    let mut hasher = Sha256::new();
    let mut stream = res.bytes_stream();
    let mut downloaded = 0u64;
    let mut last_emit = std::time::Instant::now();
    while let Some(chunk) = stream.next().await {
        let chunk = chunk.map_err(|e| {
            let _ = std::fs::remove_file(&tmp_path);
            format!("chunk: {e}")
        })?;
        file.write_all(&chunk).map_err(|e| format!("write: {e}"))?;
        hasher.update(&chunk);
        downloaded += chunk.len() as u64;
        if last_emit.elapsed() > std::time::Duration::from_millis(200) {
            let _ = app.emit("local-whisper-progress", DownloadProgress {
                model_id: model_id.clone(),
                downloaded,
                total,
                done: false,
                error: None,
            });
            last_emit = std::time::Instant::now();
        }
    }
    drop(file);

    if !expected_sha256.eq_ignore_ascii_case("TODO_VERIFY_AT_BUILD_TIME") {
        let actual = format!("{:x}", hasher.finalize());
        if !actual.eq_ignore_ascii_case(&expected_sha256) {
            let _ = std::fs::remove_file(&tmp_path);
            return Err(format!("sha mismatch: got {actual}, want {expected_sha256}"));
        }
    }

    std::fs::rename(&tmp_path, &final_path).map_err(|e| format!("rename: {e}"))?;
    let _ = app.emit("local-whisper-progress", DownloadProgress {
        model_id,
        downloaded,
        total,
        done: true,
        error: None,
    });
    Ok(())
}

#[tauri::command]
pub fn local_whisper_delete(app: AppHandle, model_id: String) -> Result<(), String> {
    let dir = models_dir(&app)?;
    let p = dir.join(format!("{model_id}.bin"));
    if p.is_file() {
        std::fs::remove_file(p).map_err(|e| format!("delete: {e}"))?;
    }
    Ok(())
}
```

   ```ts
   async function localWhisperRaw(
     blob: Blob,
     opts: TranscribeOptions,
   ): Promise<string> {
     const { localWhisperTranscribe, localWhisperCheck } = await import("./localWhisperBridge");
     const status = await localWhisperCheck();
     const wantId = opts.localWhisperModel ?? "ggml-base";
     if (!status.installed_models.includes(wantId)) {
       throw new Error(`local model ${wantId} not installed`);
     }
     const ab = await blob.arrayBuffer();
     const lang = opts.mode.language && opts.mode.language !== "auto" ? opts.mode.language : null;
     const t0 = performance.now();
     const text = await localWhisperTranscribe(wantId, new Uint8Array(ab), blob.type, lang);
     const durationMs = Math.round(performance.now() - t0);
     logDebug("transcribe-result", { status: 200, durationMs, textLength: text.length, message: `local-whisper (${wantId})` });
     return text;
   }
   ```

3. In `transcribe()`, **before** the Claude Code branch (line 280), add:
   ```ts
   if (opts.useLocalWhisper && opts.localWhisperModel !== "__cloud__") {
     try {
       logDebug("transcribe-backend", { message: `local-whisper (${opts.localWhisperModel ?? "recommended"})` });
       const raw = await localWhisperRaw(blob, opts);
       const cleanup = opts.mode.cleanup ?? "prose";
       if (cleanup === "raw" || !raw.trim()) {
         return { text: raw };
       }
       // For non-raw cleanup, route to the worker's /v1/audio/clean — but
       // the worker currently expects audio. We need a /v1/text/clean
       // endpoint OR we send the raw transcription back through Claude Code.
       // **For v0.10:** if cleanup != "raw" and useLocalWhisper, we ALWAYS
       // route the cleanup through the cloud worker by re-uploading the
       // audio. This is suboptimal but ships fast. v0.10.x will add a
       // text-cleanup endpoint.
       //
       // Simpler interim: when local Whisper succeeds AND cleanup is required,
       // skip the local result and run the full worker flow. UX cost: local
       // path only "wins" for raw modes. We document this clearly.
       //
       // Implementer choice: implement the simpler interim. Falls through to
       // the worker path below.
       logDebug("transcribe-post", {

       logDebug("transcribe-post", {
         message: "local-whisper failed, falling back to worker",
         error: (e as Error).message?.slice(0, 200),
       });
     }
   }
   ```

   **Important:** the cleanup-needed case is intentionally a fall-through. Read the inline comment carefully. v0.10 only fully replaces the cloud path for **raw** modes (Code mode, "Raw" cleanup). ## 8. Risk register

Concrete risks ordered by descending likelihood × impact.

### R1 — Audio format mismatch (HIGH likelihood, MEDIUM impact)

`MediaRecorder` on macOS WKWebView prefers `audio/mp4` (AAC), which symphonia can decode. But on some macOS versions or after WKWebView updates, the mime may default to `audio/webm;codecs=opus`, which symphonia 0.5 cannot demux. Mitigation: verify `useRecorder.ts` mime preference order on the target OS. Test on macOS 14, 15, 26 dev builds. If webm is forced, the local path will throw and fall back to the worker — user is unblocked but local mode silently fails. **Treat this as a release-blocker for ship**: if any test machine produces webm and audio_decode rejects it, add `webm` decoder via `webm-iterable` crate or pre-transcode in JS via OfflineAudioContext (~30 min addition).

### R2 — Model download UX during first use (HIGH likelihood, LOW impact if guided)

The recommended model (`ggml-base`, 142 MB) takes 30–120 s on typical home wifi. Users who toggle "Use local transcription" without first downloading a model will experience an "errorMissingModel" toast and, if they dismiss it, silent fallback to cloud. Mitigation: when the toggle flips on and no model is installed, **automatically initiate the recommended download** with a clear progress UI; don't enable the toggle until the download completes. Implementer: add this check in Phase 3, Step 3.2.

### R3 — Metal compatibility on older Macs (MEDIUM likelihood, MEDIUM impact)

`whisper-rs` with `metal` feature builds Metal shaders that require Metal 2.4+. macOS 12+ is fine; macOS 11 may fail. Our minimum supported macOS in `tauri.conf.json` is implicit from the Tauri default (macOS 10.15+). Mitigation: bump min macOS to 12 in `tauri.conf.json` `bundle.macOS.minimumSystemVersion = "12.0"`. **Implementer: add this in Phase 5, Step 5.1.** Also: on Intel Macs, the Metal path falls back to BLAS/CPU; performance will be 5–10× worse than M-series. Document in the "Beta" copy.

### R4 — Whisper hallucinations on near-silent audio (MEDIUM likelihood, MEDIUM impact)

Whisper notoriously fabricates output ("Thanks for watching!") on silent input. The renderer already does a `peakLevel` silence-gate (PillWindow.tsx:559–565) BEFORE calling transcribe — this gate covers the local path automatically. No additional risk vs cloud. **No action required**, just verify the gate still triggers in Phase 4 testing.

### R5 — Disk-space exhaustion (LOW likelihood, LOW impact)

A user clicking Download on all four models consumes ~2.4 GB. Mitigation: model size shown in the row label; deletion is one click. Acceptable as v0.10 scope.

### R6 — DMG signing breaks on Metal shader compile artifacts (LOW likelihood, HIGH impact)

`whisper-rs`'s build script invokes `xcrun -sdk macosx metal` to compile shaders. The compiled `.metallib` is statically embedded — but if the Tauri bundling step copies build artifacts in a way that re-signing fails, notarization will reject. Mitigation: do a full `pnpm tauri build` in Phase 1, Step 1.4 before any UI work. If notarization fails there, file the issue early.

### R7 — `whisper-rs` crate vendor bug (LOW likelihood, MEDIUM impact)

A new whisper-rs minor version may regress build on macOS arm64 (it has happened — see whisper-rs issues #200+ on bindgen / cmake mismatches). Mitigation: pin `whisper-rs = "=0.11.x"` to a specific point release once verified. Recheck on each Cargo.lock update.

### R8 — Token usage savings smaller than claimed (LOW likelihood, LOW impact)

Cloud cost is dominated by the cleanup LLM, not Whisper. Local Whisper saves the ~$0.0005 Whisper transcription but not the LLM cleanup. Real savings are 20–40% for typical usage, not 100%. Mitigation: don't make absolute cost claims in marketing copy. The privacy + offline + speed claims are unaffected.
